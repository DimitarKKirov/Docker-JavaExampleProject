Create Testing Environment with Docker for a locale use.

Tools

Docker for Windows - Software containerization platform.
Available at - https://www.docker.com/products/docker-desktop

Docker setup:

After (registering in docker.com) downloading and installing docker desktop,
open docker UI (inside the tray) and go to:
Settings (the cogwheel on the top)->General (the top tab) -> check the option Expose daemon on tcp://localhost:2375 without TLS
